titus.genpy.doFor¶

titus.genpy.doFor(state, scope, initLet, predicate, stepSet, loopBody)[source]¶

Helper function for constructing for loops as an expression.

Calls state.checkTime() on every iteration.

Parameters:
  • state (titus.genpy.ExecutionState) – exeuction state
  • scope (titus.util.DynamicScope) – dynamic scope object
  • initLet (callable) – initialization of for loop variables
  • predicate (callable) – function that returns True or False
  • stepSet (callable) – updating of for loop variables
  • loopBody (callable) – function that is called while predicate returns True
Return type:

None
Returns:

nothing
