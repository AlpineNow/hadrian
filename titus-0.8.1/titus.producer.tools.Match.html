titus.producer.tools.Match¶

class titus.producer.tools.Match(original, modified)[source]¶

Bases: object

Represents an expression matched by a regular expression.

original is a copy of the expression, identical to the original subexpression. modified is a copy of the expression with RegEx substitutions. groups is a dictionary of modified subexpressions.
