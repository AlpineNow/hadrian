titus.genpy.doForeach¶

titus.genpy.doForeach(state, scope, name, array, loopBody)[source]¶

Helper function for constructing foreach loops as an expression.

Calls state.checkTime() on every iteration.

Parameters:
  • state (titus.genpy.ExecutionState) – exeuction state
  • scope (titus.util.DynamicScope) – dynamic scope object
  • name (Python iterable) – new variable for each array item
  • name – array to loop over
  • loopBody (callable) – function that is called while predicate returns True
Return type:

None
Returns:

nothing
